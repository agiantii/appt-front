1. 项目概述：APPT - AI驱动的幻灯片协作编辑平台，基于React + TypeScript构建，支持实时协作、AI辅助创作和版本管理。

2. 已完成核心功能：
   1) 专业幻灯片编辑器：基于CodeMirror 6实现Markdown编辑、实时预览、语法高亮和代码片段插入
   2) AI智能辅助：实现内容生成、智能建议和AI生图功能
   3) 实时协作：基于Yjs + Hocuspocus实现多人在线编辑和光标追踪
   4) 权限管理：实现Owner/Editor/Commenter/Viewer四级角色权限控制
   5) 版本控制：实现版本历史、对比和回滚功能
   6) 知识库管理：支持文档树形结构组织和拖拽排序

3. 技术栈：React 19.2.4、TypeScript 5.8、Vite 6.2、Yjs 13.6、CodeMirror 6、React Router v7

4. 系统架构：前端采用模块化设计，包含API接口层、可复用组件和页面组件；数据库设计了用户、幻灯片、版本、评论、权限等核心表

5. 创新特色：AI驱动创作、CRDT算法实时协作、完整版本管理、细粒度权限控制

6. 后续计划：优化大文件编辑性能，完善系统测试，持续优化用户体验, 支持深色/浅色主题切换
